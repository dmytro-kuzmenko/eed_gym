# Security Policy

## Supported Versions
Security fixes will be applied to the `main` branch. Please base all reports on the latest commit before disclosing issues.

## Reporting a Vulnerability
Email security reports to dmytro.o.kuzmenko@gmail.com. Provide as much detail as possible (steps to reproduce, impact, suggested fixes). You will receive an acknowledgment within 5 business days.

Do not open public GitHub issues for security reports.
